08/02/2023 patch-- Fixed major and minor bugs and upgraded performance of the following code:

example_script_noctd.m:
	
	* fixed issues with reading EK80 raw data from different WBT and transducers. Now the code can read any 
	EK80 transducer + tranceiver combo agnostically without pre-specifying channel name.
	* fixed dependency issues while reading ES/EK60 data that required EK80 data for timesync. Added options to
	either timesync ES/EK60 data with EK80 data if needed or read ES/EK60 data as it is independently without timesync. 
	This is done by adding/"nulling" input variables to the function (see details under es60parser_sv_withoutROV.m). 
	* minor fix with plot titles
	* increased compile speed by removing the redundant range array in generate_data.m
	* removed code related to CTD stuff

	* Added start date time of 1st ping to title

	* Implemented changes in this function to reflect in generate_data.m function as well by removing the is70 indexing that created errors

	* Removed the redundant range array in generate_data.m so now the speed of compilation is greatly increased
